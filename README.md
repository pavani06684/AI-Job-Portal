AI Job Portal
An AI-Powered Job Portal using Django,Python,Bootstrap,SQLite,and AI-based Resume Analysis.

Features
->User Registration @ Login
->Role-Based Authentication(Job Seeker/Recruiter)
->Recruiter Dashboard
->Post,Edit & Delete Jobs
->Search & Filer Jobs
->Apply for Jobs
->View Application Status
->AI Resume Analyzer
->Resume Skill Extraction
->Resume Score
->Resume Improvement Suggestions
->User Profile Management
->Responsive Bootstrap

Technologies Used
->Python
->Django
->Bootstrap
->HTML5
->CSS3
->JavaScript
->SQLite
->spaCy
->pdfplumber

Project Structure
->Accounts Module
->Jobs Module
->Applications Module
->AI Resume Analyzer Module

Installation

git clone https://github.com/pavani06684/AI-Job-Portal.git
cd AI-Job-Portal
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
